# Base Project for Android Kotlin Compose Applications
## Architecture Components
This application implements the following concepts:

  * Compose
  * MVVM
  * Multi-modules
  * Navigation Component
  * ViewModel
  * StateFlow
  * Coroutines
  * Koin
  * Room
